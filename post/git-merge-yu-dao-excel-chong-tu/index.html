git merge遇到excel冲突

2022-12-14 1 min read

首先遇到冲突, 很自然的想到用

git mergetool

结果当然是merge失败

找了一些参考:
一种思路是云风的Excel转文本
https://blog.codingnow.com/2014/11/excel.html
但只是提高可读性, 对于处理merge意义不大, 另一种思路是开发MergeTool
https://github.com/ThisIsRone/ExcelMergeTool

这些方案学习成本都比较高, 对于简单的merge, 有一种原始但简单的方法:

  • 把两份冲突的excel, 用不同文件名保存在本地
  • 打开两个文件, 手动合并后提交
